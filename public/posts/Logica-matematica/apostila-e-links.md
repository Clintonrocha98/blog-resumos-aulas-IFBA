# Glossário

Alguns símbolos que você vai ver na aula e provavelmente não sabe o significado: 
- ∈ - pertence
- ∉ - não pertence
- ⊂ - contido
- ⊄ - não contido 
- ⊃ - contem 
- ⊅ - não contem
- `|` - Em conjuntos, o símbolo `|` é geralmente lido como "tais que" ou "onde", indicando a descrição da propriedade que os elementos do conjunto devem satisfazer. Por exemplo, na expressão "{x | x é um número natural par menor que 12 e maior que 0}", o `|` é lido como "tais que" ou "onde".
- ∪ - união 
- ∩ - interseção 
- ∨ - disjunção lógica ou "ou", indica que pelo menos uma das afirmações conectadas é verdadeira
- ∧ - conjunção lógica ou "e", indicar que pelo menos uma das afirmações conectadas é verdadeira, é usado para indicar que todas as afirmações conectadas devem ser verdadeiras.,

## Conjuntos Numéricos

Os **conjuntos numéricos** reúnem diversos conjuntos cujos elementos são números. Eles são formados pelos números naturais, inteiros, racionais, irracionais e reais. O ramo da matemática que estuda os conjuntos numéricos é a Teoria dos conjuntos.

## Diagrama de Venn

<iframe src="https://drive.google.com/file/d/1rydGT1SlB8YXa48100iVhX_bBIrevRa1/preview" width="640" height="480" allow="autoplay"></iframe>

O diagrama de Venn é uma forma gráfica que representa os elementos de um conjunto. Para fazer essa representação utilizamos formas geométricas.

Para indicar o conjunto universo, normalmente usamos um retângulo e para representar subconjuntos do conjunto universo empregamos círculos. Dentro dos círculos são incluídos os elementos do conjunto.

# Apostila sobre Conjunto

## Noções básicas

Conceitos primitivos são teorias aceitas sem demonstrações.

Na teoria dos conjuntos, conjunto, elemento de um conjunto e pertinência de um elemento em um conjunto, são conceitos primitivos.

A ideia de conjunto pode ser considerada como a ideia de coleção.

Como exemplo de conjunto tem: os alunos de uma escola formam um conjunto e os alunos são os elementos, outro exemplo é a coleção de livros de uma biblioteca que forma um conjunto onde cada livro é o elemento.

Outros exemplos:

a) Conjunto dos números 0,1,2,3,4,5...
b) Conjunto dos números impares 1,3,5,7,9...
c) Conjunto das figuras geométricas.

Representação de um conjunto

Podemos representar conjunto de várias formas, por exemplo, vamos representar o conjunto P formados pelos elementos 2,4,6,8,10.

Enumerando os elementos:
P = {2,4,6,8,10}

Considerando uma propriedade dos elementos
P = {x|x é um número natural par menor que 12 e maior que 0}

Através do diagrama de Venn

Relação de pertinência (∈ pertence, ∉ não pertence)

Exemplo:
- 2 pertence a P, escrevemos 2 ∈ P.
- 3 não pertence a P, escrevemos 3 ∉ P.

## Igualdade de conjunto

Dois conjuntos A e B são iguais (A = B) se e somente se A ⊂ B e B ⊂ A.

Exemplos:

- M = {x|x é um número natural múltiplos de 3 menores que 10}
- N = {0,3,6,9}
- M = N.

## Conjunto vazio, unitário e universo

### Conjunto vazio é o conjunto que não tem elementos

Exemplo:
- V = {x|x é um número primo menor que 2}
- Podemos escrever V = ∅.

### Conjunto unitário é o conjunto formado por um único elemento

Exemplo:

- N = {x|x é um número natural primo par}
- Escreve-se N = {2}

### Conjunto Universo é aquele ao qual pertencem todos os elementos relacionados ao determinado estudo

Exemplo:

- U = {x|x é a quantidade de pessoas que estão com dengue}

## Subconjunto de um conjunto

Um conjunto B é subconjunto de A se e somente se todos os elementos de B também é elemento de A.

Exemplo:

- A = {0,2,4,6,8,10}
- B = {0,4,8}

Dizemos que B é subconjunto de A, e escrevemos B ⊂ A.

Podemos ler B está contido em A.

Se escrevermos A ⊃ B, podemos ler A contém B.

# Operações com conjuntos
## União de conjuntos

Dados os conjuntos A e B, indicaremos por A ∪ B ao conjunto formado por todos os elementos de A ou de B. Assim temos: A ∪ B = {x|x ∈ A v x ∈ B}.

Exemplos

a) Dados os conjuntos A = {1,2,3} e B = {5,6,7}, temos A ∪ B = {1,2,3,5,6,7}.

b) Dados os conjuntos A = {1,2,3,4} e B = {3,4,5,6,7}, temos A ∪ B = {1,2,3,4,5,6,7}

c) Dados os conjuntos A = {3,6,9} e B = {1,2,3,4,5,6,7,8,9}, temos A ∪ B = {1,2,3,4,5,6,7,8,9}.

## Intersecção de conjuntos

Dados os conjuntos A e B, indicaremos por A∩B ao conjunto formado por todos os elementos de A e de B. Assim temos: A∩B = {x|x∈A∧x∩B}.

Exemplos

a) Dados os conjuntos A = {1,2,3} e B = {5,6,7}, temos A∪B = { }.

b) Dados os conjuntos A = {1,2,3,4} e B = {3,4,5,6,7}, temos A∪B = {3,4}

c) Dados os conjuntos A = {3,6,9} e B = {1,2,3,4,5,6,7,8,9}, temos A∪B = {3,6,9}.

# Diferença entre conjuntos

Dados os conjuntos A e B, a diferença de A e B é o conjunto formado pelos elementos de A que não são de B. Assim temos A – B = {x|x∩A e x∈B}.

Exemplos:

a) Dados os conjuntos A = {1,2,3} e B = {5,6,7}, temos A – B = {1,2,3}.

b) Dados os conjuntos A = {1,2,3,4} e B = {3,4,5,6,7}, temos A – B = {1,2}

c) Dados os conjuntos A = {1,2,3,4,5,6,7,8,9} e B = {3,6,9}, temos A – B = {1,2,4,5,7,8}.

## Conjunto dos números racionais

Números racionais são números que representam uma razão entre dois números inteiros, sendo o segundo diferente de zero.

Indicamos por:

<img src="https://latex.codecogs.com/svg.image?\mathbb{Q}={x|x=\frac{a}{b}\text{,com}a\in\mathbb{Z}\wedge&space;b\in\mathbb{Z^{*}}}"/>

Os números racionais podem ser representados por decimais finitos (podendo ser inteiro) e decimais infinitos (dízima periódica).

Exemplos:

Número com representação decimal finita:

<img src="https://latex.codecogs.com/svg.image?\text{a)}\frac{4}{10}=0,4"/> 

Número com representação decimal finita:

<img src="https://latex.codecogs.com/svg.image?\text{b)}\frac{32}{16}=2"/>

Número com representação decimal infinita, dízima periódica:

<img src="https://latex.codecogs.com/svg.image?\text{c)}\frac{2}{6}=0,333..."/>

## Conjunto dos números irracionais

O conjunto dos números irracionais é formado por todos os números que não podem ser expressos na forma:

<img src="https://latex.codecogs.com/svg.image?\frac{a}{b}\text{ com&space;a}\in\mathbb{Z}\wedge\in\mathbb{Z^{*}}"/>

Assim os números irracionais são aqueles cujo representação decimal é infinita e não é uma dízima periódica.

Exemplos:

a) A raiz quadrada de um número natural que não é quadrado perfeito:

<img src="https://latex.codecogs.com/svg.image?\sqrt{2},\sqrt{3},\sqrt{5},\sqrt{12},\sqrt{15}."/>

b) A raiz cúbica de um número natural que não é cubo perfeito:

<img src="https://latex.codecogs.com/svg.image?\sqrt[3]{2},\sqrt[3]{4},\sqrt[3]{5},\sqrt[3]{10}."/>

c) 0,2002000200002... observemos que a quantidade de zeros que sucede o algarismo 2 aumenta indefinidamente.

## Conjunto dos números reais

O conjunto dos números reais é a união entre os conjuntos dos números racionais e o conjunto dos números irracionais. 

<img src="https://latex.codecogs.com/svg.image?\mathbb{R}=\mathbb{Q}\cup\mathbb{I}"/>

(onde Q é o conjunto dos números racionais e I o conjunto dos números irracionais)
